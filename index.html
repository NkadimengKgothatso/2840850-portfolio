Latest Project

Project: City Noise Mapper A mobile app that crowdsources audio clips from users and uses AI to classify sounds like traffic, nightlife, or construction. It visualizes noise levels on an interactive city map, helping residents find quieter routes, study spots, or rest times. Built with Python (Librosa + PyTorch), Google Maps API, and React Native, it turns the chaos of urban noise into actionable insights.
